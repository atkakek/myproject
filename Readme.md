# Verzokezelés
## Helyi repository létrehozása

- helyi repo inicializálása
    >git init
- ellenőrzés
    >git status
- előkíszitjuk az uj verziot, beindexelés 
    >git add .
- ellenőrzés: user, email
    >git config user.name

    >git config user.email
- verzio eltarolása
    >git commit -m "valtozasok"
